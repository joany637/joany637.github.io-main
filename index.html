Around - Perfil de Explorador

Fue desarrollado usando HTML, CSS y JavaScript básico para construir una interfaz visualmente atractiva y funcional que simula un perfil personal editable con contenido dinámico.

Triple Espresso

Es una página web de una cafetería (Coffee Shop) desarrollada con HTML y CSS, diseñada para mostrar un sitio visualmente atractivo de un negocio de café. El proyecto incluye una estructura de contenidos con secciones como menú, imágenes y otros elementos visuales que representan una experiencia típica de una cafetería moderna en la web. Está organizada con estilos propios para lograr una presentación limpia, funcional y representativa del concepto de un Coffee Shop.
